# liver-cancer-prediction
In this project, an AI-based system is developed to predict liver cancer stages (Stage I–IV) using CT scan images. Deep features are extracted using DenseNet121 CNN and classified using an ensemble of SVM (RBF), Random Forest, and XGBoost models.

##Tools Used
-Python
-TensorFlow / PyTorch
-Scikit-learn
-OpenCV
-Jupyter Notebook / VS Code

##Models & Techniques
-DenseNet121 (CNN) – Feature Extraction
-SVM (RBF Kernel) – Classification
-Random Forest – Classification
-XGBoost – Classification
-Ensemble Voting – Final Prediction

##Key Features
-CT scan image preprocessing and normalization
-Deep feature extraction using CNN
-Multiple ML classifiers for robust prediction
-Ensemble learning to reduce error and improve accuracy
-Accurate liver cancer stage prediction (Stage I–IV)

##Application
-Medical image analysis
-Decision support for doctors
-Early and accurate liver cancer staging
